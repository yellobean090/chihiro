Build a complete, single-file demo website (HTML + CSS + JS, no frameworks) for CHIHIRO — a bakehouse and café located at 3, Mahatma Gandhi Marg, Sadar Bazaar, Cantonment, Lucknow, Uttar Pradesh 226002. The café is female-led, run by a young couple, inside a war memorial campus surrounded by lush cantonment greenery. The name "Chihiro" is Japanese for "a thousand questions" — evoking curiosity, wonder, and quiet discovery. The overall aesthetic is Studio Ghibli meets Parisian bakehouse: warm, romantic, botanical, soft-feminine, nature-drenched, deeply curated.

=== TECH REQUIREMENTS ===

- Single HTML file. Vanilla HTML + CSS + JavaScript only. No frameworks, no Tailwind, no Bootstrap.
- Load GSAP 3 from CDN: https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/gsap.min.js
- Load GSAP ScrollTrigger plugin from CDN.
- Load Google Fonts via <link> tag.
- Where real photos would go, use styled <div> placeholders with CSS gradients in the brand palette, layered with botanical SVG overlays, so the site looks complete and beautiful without actual images.
- All code in one file. Fully functional. No broken sections.

=== GLOBAL EFFECTS IMPLEMENTATION NOTES ===

Parallax: Use GSAP ScrollTrigger with scrub:1 on the hero image, ambience section, and any section background layers.

Entrance Reveals: Every section heading uses a clip-path reveal: the text is inside overflow:hidden, and on ScrollTrigger entry, inner element goes from translateY(100%) → 0. Body content uses opacity + translateY stagger.

Scroll Trigger Actions: Every section (About, Menu, Showcase, Ambience, Info, Reviews, Footer) has at least one ScrollTrigger-activated animation.

Floating Objects: 4–6 SVG botanical elements scattered across the page (not just hero). Each has a unique CSS @keyframes animation with translateY(-15px to 15px) and slight rotation, running at different durations (7s, 11s, 14s, 8s) on infinite alternate loops.

3D Motion: Ambience section large card uses CSS perspective(800px) + JS mousemove → rotateX/Y calculation. Smooth lerp applied to the rotation values.

Micro Interactions:
- Menu price values spring-bounce in on hover
- Tab indicator slides with GSAP
- Button hover: slight scale(1.04) + shadow expansion
- Star rating fill animation on review card entrance
- Cursor transformations on different hover targets

Mobile Friendly:
- All grid layouts collapse to single column at <768px
- Carousel becomes touch/swipe-scrollable
- Custom cursor disabled on touch devices
- Hero splits to single column stacked layout
- Font sizes scale down with clamp()
- Floating elements reduced to 2 on mobile (performance)

Accessibility:
- All images/placeholders have descriptive alt text
- All buttons and interactive elements are keyboard navigable
- Visible focus rings on all focusable elements (2px terracotta outline, 2px offset)
- prefers-reduced-motion: wrap all GSAP animations in a matchMedia check — if reduced motion preferred, disable transitions and show content immediately
- Semantic HTML: <header>, <main>, <section>, <nav>, <footer>, <article> used correctly
- aria-label on all icon-only buttons
- Color contrast meets WCAG AA (all text over background combinations)
- Wheelchair accessibility info prominently featured in Info section

Performance:
- Loader hides content until ready (prevents FOUC)
- GSAP animations use will-change: transform only when needed
- CSS animations use transform and opacity only (GPU-composited)
- No layout-triggering properties animated (no width/height/top/left)

Output: A single, complete, copy-pasteable HTML file. Every section fully built. No placeholder "// add code here" comments. No broken layouts. The site must look premium, feminine, editorial, and emotionally warm — like a high-end European café that happens to be in Lucknow.
